## Getting Started 🚀

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/calculator-design.git
    ```
2. Navigate to the project directory:
    ```bash
    cd calculator-design
    ```
3. Open `index.html` in your web browser to view the calculator.
